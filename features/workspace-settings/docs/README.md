# Настройки workspace (Workspace Settings)

> 2026-05-23 · статус: полностью реализован (бэкенд API и интерактивный фронтенд управления участниками, ролями, ссылками-приглашениями и разрешенными доменами).

---

## Назначение

Модуль предназначен для администрирования рабочего пространства (workspace): управление списком участников команды, назначение и смена ролей, отправка приглашений по электронной почте, генерация пригласительных ссылок и настройка ограничений по доменам электронной почты.

**Маршрут:** `/team` (страница «Команда»)
**Связанные таблицы:** `workspace_members`, `workspace_invitations`, `workspace_allowed_domains`

### Роутинг (App Router)
- `/team` — рендерит страницу `TeamPage` (вызывает `TeamManagementView` из модуля настроек воркспейса).

---

## Данные

### Таблицы базы данных

- **`workspace_members`** — список всех пользователей, привязанных к текущему владельцу воркспейса (`owner_id`), со статусом (`active`, `suspended`, `invited`) и аудитом времени.
- **`workspace_invitations`** — активные email-приглашения в команду со сроком жизни и статусом.
- **`workspace_allowed_domains`** — доменные ограничения для регистрации в данном воркспейсе.

---

## API (Next.js Route Handlers)

Все эндпоинты настроек воркспейса размещены под роутом `/api/team/*` и используют серверную проверку прав (`team.manage`):

- `GET /api/team/members` — Получить список участников воркспейса.
- `POST /api/team/members` — Назначить роль участнику воркспейса.
- `DELETE /api/team/members` — Снять роль с участника (запрещено для locked-ролей).
- `GET /api/team/domains` — Разрешенные почтовые домены для воркспейса.
- `POST /api/team/domains` — Зарегистрировать новый разрешенный домен.
- `DELETE /api/team/domains` — Удалить разрешенный домен.
- `GET /api/team/invitations` — Список отправленных и не принятых email-приглашений.
- `POST /api/team/invitations` — Отправить новое приглашение на email.
- `DELETE /api/team/invitations` — Отозвать отправленное приглашение.
- `GET /api/team/invite-link` — Получить параметры действующей публичной пригласительной ссылки.
- `POST /api/team/invite-link` — Сгенерировать или сбросить публичную пригласительную ссылку.
- `GET /api/team/overview` — Сводные показатели по воркспейсу (количество человек, роли, активность).
